## Comment lancer l'appli en local

### Lancer le backend

`cd Billed-app-FR-Back`

`npm run run:dev`

Accéder à l'API :

L'api est accessible sur le port 5678 en local, c'est à dire http://localhost:5678

### Lancer le frontend

`cd Billed-app-FR-Front`

`live-server`

Puis allez à l'adresse : http://127.0.0.1:8080/
